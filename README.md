# X-Large Brain (XLXL) 🧠

Personal engineering standards and task execution framework for high-efficiency AI-assisted development.

## 📌 Overview
This repository serves as the central "source of truth" for my engineering workflows, mandates, and task archives. It ensures a consistent, disciplined, and transparent approach across all development environments (personal and professional).

## ⚖️ Core Standards: XLXL.md
The heart of this repository is [XLXL.md](./XLXL.md), which defines:
- **Language Consistency**: All standards and logs are maintained in English.
- **Task Hierarchy**: Centralized task tracking under `~/xlxl-tasks/`.
- **Approval Gate**: Mandatory user consent before any external system or file modifications.
- **Artifact Governance**: Structured archival of findings, progress, and verification results.

## 🛠 Usage
This repository is automatically integrated into my AI CLI sessions (Gemini CLI, Kiro, etc.) by configuring them to read the `XLXL.md` file as a primary instructional context.

## 📂 Structure
- `XLXL.md`: The core engineering mandates.
- `README.md`: Project landing page.
